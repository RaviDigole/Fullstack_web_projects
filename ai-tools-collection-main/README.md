# 🤖 AI Tools Collection

A curated collection of **AI-inspired web applications (frontend simulations)** built using **HTML, CSS, and JavaScript**.
This repository showcases the design and behavior of modern AI tools such as chatbots, generators, analyzers, and assistants.

> ⚡ Focus: UI/UX + Logic Simulation (No backend AI)

---

## 🚀 Live Demo

👉 https://aitools-krushnataur.vercel.app
*(Access all AI tools from a single landing page)*

## 🧩 Tech Stack

* HTML5
* CSS3 (Flexbox / Grid)
* Vanilla JavaScript
* Chart-less lightweight UI
* Online image sources (Unsplash)
* Predefined datasets for AI simulation
